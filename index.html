  • Guide d'utilisation
  • Règles de comptage
  1. Choisissez l'option 'Coller le texte dans le champ' si vous souhaitez utiliser le champ de saisie
  2. Choisissez l'option 'Sélectionner un fichier' si vous souhaitez lire le nombre de mots contenus dans un fichier
  3. Téléchargez des fichiers txt. Les autres types de fichiers (.docx, .pdf, etc) ne sont pas lisibles et le nombre de mots pourrait être inexact
  4. Cliquez sur le bouton 'démarrer le comtage' pour compter
  1. Un mot simple compte pour 1 mot
  2. Les signes de ponctuation ne sont pas comptés
  3. Un mot complexe compte pour 1 mot (ex: tire-bouchon)
  4. Une lettre avec apostrophe compte pour 1 mot (ex: L')
  5. Un nombre entier compte pour 1 mot
  6. Un nombre décimal compte pour 1 mot
  7. Les symboles monétaires ne sont pas comptés (ex: $, £)
  8. Les symboles mathématiques ne sont pas comptés
  9. Une adresse email compte pour 1 mot
  10. Les balises HTML ne sont pas comptées
